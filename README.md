# Node js boilerplate

## How to use
  
- clone this repo
- run
  
        npm install


## packages Installed

- debug
- bootstrap
- jquery
- ejs
- chalk (for terminal style)
- eslint (for styling)
- nodemon (automating the server)
- passport (auth)
- morgan (middleware)
- body-parser (body parsing middleware.)
- express
- express-session (session middleware)
- path

## Starting the server

    npm start
